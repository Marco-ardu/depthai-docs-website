防水外壳
=====================

OAK-1 和 OAK-D 外壳
**************************

OAK-1 和 OAK-D 外壳不防水, 这是因为他们的外壳没有防水胶圈 (只有PoE系列有, 如下图).

IP67 级 PoE 外壳
*************************

PoE 设备 ( `OAK-D-POE <https://www.oakchina.cn/product/oak-d-poe/>`__ 和 `OAK-D-POE <https://www.oakchina.cn/product/oak-1-poe/>`__ ) 附带 **IP67** 防护等级. `IP code <https://en.wikipedia.org/wiki/IP_Code>`__
告诉您设备是否防尘/防水. IP67 意味着该设备可以 **防止灰尘进入**，并且可以 **防止 15 厘米到 1 米深的水浸入**.

PoE外壳有胶圈可以让他防水.

.. image:: /_static/images/waterproof/poe_gasket.jpeg
